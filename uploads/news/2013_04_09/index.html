Importing Accounts

You spoke, we listened. importing accounts from a CSV file is now available.

To get to the import page go to AdBooker settings -> accounts -> import

Export your accounts list from your accounting software into a CSV file, paste the contents of the CSV file into the "import" box.

AdBooker then looks up the existing records and tries to match them, showing you stats

Selecting records (marking the checkbox) and clicking save will then override existing records (those in blue and checked) and add all new records to the mix

All the checked records will get the options on the right (publications / Status)
