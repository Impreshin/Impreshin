Placing, Colours and Sub Placing

The placing section got a bit of a makeover recently. Sub placings can be added.

There is an option to force a colour now, this is usefull if you still make use of different colour tarrifs for the various placing sections. if there is no forced colour then the user will be provided with a drop down when capturing a booking to select which colour is available.

Should your publication only use full colour it would be advisable to just set all your placing / sub placing as "Full Colour" as to limit confusion
