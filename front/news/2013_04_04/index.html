New columns for list views

Some more columns for you to choose from:

Column Description Eg.
Date This was always there but its been changed to just the date the record was added 2013-03-06
Date\Time The date and time the record was added 2013-03-06 10:07:14
Last Changed Date of the last change to the record, this includes editing / deleting/ checking / adding the record to a page etc 2013-03-06
Last Changed DT Date and time of the last change, the above is just the date 2013-03-06 10:07:14
